## 📝 部署到你自己的 GitHub

1. 在 GitHub 上新建一个 Public 仓库(比如叫 `my-aiyue-sources`)
2. 把本仓库所有 .ibs 文件**直接拖到 GitHub 网页上传**(支持批量,数百个文件可一次拖完)
3. 等上传完成,记下仓库 URL: `https://github.com/<你的用户名>/<仓库名>`
4. 在爱阅书香 App 的同步功能里输入这个 URL
